bildr.co

August, 2015

2-person project


Written in Ruby, JavaScript, and Materialize HTML

Submission for Riot Games API Challenge 2.0

Based on popular game League of Legends, bildr.co calculates how effecient an item set is, based on the gold effeciency of each item. bildr.co allows a user to upload their own item set, modify their item set in our item set builder, and finally download an improved item set to use in game. bildr.co also analyzes several key features of the item set and relays that back to the user.

I implemented the drag-and-drop ability of the item set builder, along with the search and filtering of items. I also worked on general front-end development of the site.

Amazing NASA Universe Simulator

audiopod.me

April, 2015

4-person project


Written in Python (Flask), JavaScript, and Bootstrap HTML

Submission for LA Hacks 2015

audiopod.me is a "21st-century jukebox" that allows a host to create a playlist from YouTube based on guests' suggestions. A host creates a room and guests may join the room to add their own suggestions of what to play next. audiopod.me takes those suggestions and creates a playlist where the host can then decide to keep or delete the song from the playlist.

I implemented the front-end of the website as well as some Python/JavScript scripting to interact with the back-end when creating playlists with YouTube.
